Tree-select with Drag-drop

Two bs-tree-select instances whose selected chips are draggable between them. Each chip is custom light-DOM content projected via slot="chips" with a cdkDragHandle; each tree-select sits in its own cdkDropList, and the two lists are connected so a chip dropped on the other list transfers (via transferArrayItem) — moving that node's selection across.
