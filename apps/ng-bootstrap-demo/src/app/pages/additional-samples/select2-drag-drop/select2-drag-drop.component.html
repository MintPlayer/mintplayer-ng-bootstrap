Select2 with Drag-drop

Combine bs-select2 with Angular CDK's cdkDropList + cdkDrag to let users reorder selected items by dragging. Mark the host as a drop list, mark each rendered item template with cdkDrag, then mutate selectedItems from cdkDropListDropped using moveItemInArray.

Basic usage

This example demonstrates the select2 component with the ability to drag and reorder the selected items.
