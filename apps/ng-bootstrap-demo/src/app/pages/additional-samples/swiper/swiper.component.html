Swiper

The swiper is the touch-friendly gesture layer that powers bs-carousel: pointer/touch swipes advance slides while keyboard and indicator controls remain available. Use it by simply enabling the carousel's [animation]="'slide'" mode and projecting images with *bsCarouselImage — the swipe gestures are wired automatically.

Basic usage

The swiper is used in the carousel component. View this page on a phone, or open DevTools F12 in order to enable the mobile viewer.
