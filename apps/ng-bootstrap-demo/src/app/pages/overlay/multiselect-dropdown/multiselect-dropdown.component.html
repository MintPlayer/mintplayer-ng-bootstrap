Multiselect dropdown

bs-multiselect renders a dropdown that allows multiple items to be selected via a checklist. The selection is exposed through the [(selectedItems)] two-way binding. Items are projected through *bsItemTemplate; the trigger label is customised via *bsButtonTemplate, and optional *bsHeaderTemplate / *bsFooterTemplate slots can host a search input or actions.
