Treeview

Data-driven recursive tree. Pass a TreeNode[] via [items], optionally provide an iconResolver that returns SVG markup for each node's iconKey, and bind [(expandedIds)] / [(selectedIds)] if you need to control the state.

Keyboard shortcuts
  • ↑ / ↓ — move focus to previous / next visible row
  • → — expand collapsed folder, or jump to first child
  • ← — collapse expanded folder, or jump to parent
  • Home / End — first / last visible row
  • Enter — toggle selection (and expand/collapse if folder)
  • Space — toggle selection on focused row
