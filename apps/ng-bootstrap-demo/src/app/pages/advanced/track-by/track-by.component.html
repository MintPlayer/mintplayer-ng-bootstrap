Track By

Since angular 17, you can use the control-flow syntax. This makes using bsTrackBy: 'id' obsolete.
  • Open devtools F12 and find and select the ul inside the bs-list-group
  • Click the button. Note that only the newly added bs-list-group-item is recreated
@for (person of people; track person.id) { {{ person.firstName }} {{ person.lastName }} }
