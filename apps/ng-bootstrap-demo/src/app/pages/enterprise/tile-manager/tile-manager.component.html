Tile manager

Drag a tile by its header to move it; drag a side / bottom / corner adorner to resize. Neighbouring tiles get pushed and gravity-compacted upward in real time. Layout auto-persists to localStorage — reload to verify.

Basic usage

Keyboard shortcuts
  • Tab — focus the tile board
  • ↑ / ↓ / ← / → — move focus between tiles (row-major)
  • Home / End — first / last tile
  • M — enter move mode on the focused tile
  • In move mode: ↑ / ↓ / ← / → nudge the tile, Shift + arrow resizes it
  • Enter — commit · Esc — cancel and revert
